



#### Занятие 5 << Циклические конструкции >>


##### Цикл с предсусловием

######  Конструкция ` while { } `

 
- Общиий вид :

```
while (условие)
{
тело цикла;
}

```



##### Цикл с  постусловием

######  Конструкция `do { } while `


- Общий  вид :


```
do
{
тело цикла
} while (условие);

```
